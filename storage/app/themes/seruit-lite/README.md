# Tema Seruit Lite 1.0.0 untuk OpenSID

Selamat datang di **Tema Seruit Lite**, versi standar dari tema Seruit yang dirancang khusus untuk memberikan tampilan website desa yang modern, bersih, dan profesional. Tema ini dibangun dengan teknologi **Tailwind CSS v3** untuk memastikan performa yang sangat ringan dan responsif di semua ukuran layar.

## 🌟 Fitur Utama Versi Lite

*   **Desain Modern & Sharp:** Antarmuka dengan sudut tajam (lancip) yang memberikan kesan gagah dan rapi.
*   **Warna Hijau Teal Ikonik:** Menggunakan skema warna hijau yang segar, melambangkan pertumbuhan dan kepercayaan.
*   **Fully Responsive:** Tampilan website tetap cantik saat dibuka melalui HP, Tablet, maupun Laptop.
*   **Layout Dua Kolom:** Halaman artikel yang dilengkapi dengan sidebar widget untuk memudahkan navigasi pengunjung.
*   **Widget Jadwal Sholat:** Menampilkan waktu ibadah secara otomatis berdasarkan pilihan Kabupaten/Kota di seluruh Indonesia.
*   **Aparatur Desa Slider:** Menampilkan daftar perangkat desa dengan sistem slide standar yang fungsional.
*   **Status Kehadiran:** Terintegrasi dengan sistem rekam kehadiran OpenSID.

## 🚀 Panduan Instalasi

1.  Unduh file ZIP **Tema Seruit Lite**.
2.  Masuk ke direktori instalasi OpenSID Anda melalui File Manager (cPanel) atau FTP.
3.  Cari folder `desa/themes/`.
4.  Upload dan ekstrak file ZIP tersebut di sana. Pastikan nama foldernya adalah `seruit-lite`.
5.  Masuk ke **Admin OpenSID > Admin Web > Tema**.
6.  Pilih **Seruit Lite** dan klik tombol **Aktifkan**.
7.  Disarankan untuk melakukan **Bersihkan Cache** pada menu Pengaturan Aplikasi.

## ⚙️ Panduan Pengaturan Tema

Seluruh konfigurasi dapat diakses melalui menu **Pengaturan > Tema > Pengaturan Tema (Seruit Lite)**.

### 1. Palet Warna
Versi Lite secara default dikunci pada warna **Hijau ke Teal**. Anda tetap bisa melihat pilihan warna premium lainnya di menu pengaturan sebagai referensi sebelum melakukan upgrade ke versi PRO.

### 2. Sambutan Kepala Desa
Anda dapat mengisi sambutan khusus. Jika kolom ini **dikosongkan**, tema akan secara otomatis menampilkan sambutan cerdas yang nama dan jabatannya akan selalu terupdate mengikuti data Kepala Desa yang aktif di database penduduk.

### 3. Widget Jadwal Sholat
Pilih nama Kabupaten atau Kota Anda melalui dropdown yang tersedia. Jadwal sholat akan muncul secara otomatis di halaman beranda mengikuti zona waktu lokasi Anda.

### 4. Fitur PRO [Terkunci]
Bagian pengaturan bertanda **[PRO]** seperti SEO Schema, Google Analytics, AdSense ID, dan Proteksi Klik Kanan tidak akan memberikan efek pada versi Lite ini.
